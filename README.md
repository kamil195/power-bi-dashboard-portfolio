# Power BI Dashboard Portfolio

A collection of Power BI coursework dashboards organized for portfolio review.

## Current source files

The original archive contains five `.pbix` files:

- Assignment no 3.pbix
- classactivity.pbix
- Quiz (1).pbix
- Quiz.pbix
- Activity 2nd.pbix

These filenames are classroom-style names. Each dashboard should be renamed after opening it and confirming its actual topic.

  project-02/

## Portfolio standard

For every dashboard:

1. Rename the PBIX file according to its real subject.
2. Add the source dataset when redistribution is allowed.
3. Export a clean dashboard screenshot.
4. Document the Power Query transformations.
5. List important DAX measures and KPIs.
6. Summarize the main insights.
7. State assumptions and data-quality limitations.

## Important note

Power BI files cannot be previewed directly on GitHub. Screenshots and concise documentation are therefore essential for recruiters and reviewers.
